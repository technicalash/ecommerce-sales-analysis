# Sales Data Analysis Project
### Overview

This project analyzes sales data to uncover trends, product performance, regional patterns, and profitability insights.

### Tools Used
Python
Pandas
Matplotlib
Seaborn
### Key Insights
Sales show seasonality, peaking in May and dropping in February.
Cameras and monitors drive the highest revenue and profit.
Smartwatches have high demand but lower revenue contribution.
Electronics is the top-performing category.
Profit depends more on sales than quantity, highlighting the importance of margins.
### Visualizations
Sales trends over time
Product and category performance
Regional analysis
Sales vs Profit relationships
Correlation heatmap
### Conclusion

The analysis highlights that focusing on high-margin products and region-specific strategies can improve business performance.

The dataset used in this project is publicly available on Kaggle
